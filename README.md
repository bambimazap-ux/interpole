# פרויקט הנגשת סקירות אינטרפול 2021-2025 - חטיבה לזיהוי פלילי (מז"פ)

מערכת דינמית להנגשת סקירות האינטרפול בתחומי הדנ"א הפורנזי וטביעות האצבע עבור קציני וחוקרי החטיבה לזיהוי פלילי.

## מבנה הפרויקט
* **תרגומים וסיכומים:** חלוקה מובנית לפרקים הכוללת תקצירי מנהלים בעברית והרחבות טכניות.
* **ממשק משתמש:** אתר אינטראקטיבי דינמי (Single Page App) מעוצב ומותאם לעבודה ברשת פנימית או מקוונת.
* **מולטימדיה:** נגני אודיו ווידאו מובנים להשמעת והצגת סקירות שהופקו בעברית (למשל מ-NotebookLM).
* **קישור ישיר למקור:** ניווט מתוך הממשק ישירות לעמוד הרלוונטי בדו"ח המקורי באנגלית.

## הרצה מקומית
ניתן לפתוח את הקובץ `index.html` ישירות בדפדפן (Chrome / Edge) ללא צורך בהתקנות או בשרת מקומי.
